## URL 规范

遵循项目统一埋点规则（参见 `埋点/埋点规则.md`）：

- 格式：`root/pageName/actionName`
- root 类型：`click`（点击）、`event`（自定义事件）
- pageName：`dashboardPage`
- actionName：小驼峰命名

## 参数规范

- 所有 key 和 value 使用小驼峰命名
- STRING 类型参数传原始值
- INT 类型参数传数字
