# 试卷讲解 产品稿

> 版本：v1.0 · 2026-04-15

---

## 1. 页面概述

### 功能说明

试卷讲解列表页展示可观看 AI 视频讲解的试卷资源，支持按科目、版本、年级三维筛选，用户选择试卷后进入该试卷的视频讲解详情页。

定位是"精品试卷资源库"——汇集各版本教材对应的期中/期末试卷，由 AI 龙老师逐题讲解，适合考前复习和查漏补缺。

功能边界：本页面只负责试卷列表的展示和筛选，不包含视频播放和题目解析（由视频讲解详情页承载）。

---

## 2. 筛选功能

### 功能说明

页面顶部三个筛选维度，支持快速定位目标试卷。筛选结果实时更新，无需手动刷新。

### 交互流程

1. **展开筛选**：点击任一筛选项（科目/版本/年级）→ 展开下拉列表 + 显示遮罩层
2. **选中选项**：点击下拉列表中的选项 → 列表收起 → 试卷列表自动刷新
3. **切换筛选项**：已展开一个筛选项时点击另一个 → 当前收起，新的展开
4. **收起筛选**：点击已展开的筛选项或遮罩层 → 下拉列表收起

### 业务规则

- **科目选项**：数学、语文、英语
- **版本选项**：不限、人教版、北师版、沪教版、苏教版、青岛五四版、西师版
- **年级选项**：三年级、四年级、五年级、六年级
- **默认值**：科目默认"数学"，版本默认"不限"，年级根据用户学生信息自动填充
- **组合筛选**：三个筛选项独立生效，交叉过滤
- 支持 URL 参数 `subject` 和 `grade` 预设筛选值

### 数据说明

> **筛选联动**：三个筛选项独立过滤，不做级联（即不会因为选了"语文"就只显示语文版本的教材版本列表）
> **默认年级**：优先取用户绑定学生的年级信息，无学生信息时默认"五年级"

### 异常场景

| 场景 | 处理方式 |
|------|---------|
| 筛选结果为空 | 显示"暂无符合条件的试卷，请调整筛选条件" |
| 同时点击多个筛选项 | 同一时间只有一个筛选项展开 |

## 3. 试卷卡片列表

### 功能说明

展示筛选后的试卷列表，每张卡片包含试卷封面、标题、学习人数等信息，点击进入视频讲解详情页。

### 交互流程

1. **浏览列表**：上下滚动浏览试卷卡片
2. **进入详情**：点击整张卡片或"视频讲解"链接 → 跳转到该试卷的视频讲解详情页，携带试卷 ID、标题、科目参数

### 业务规则

- 推荐试卷显示"推荐"标签
- 试卷封面根据科目自动匹配颜色主题：数学（橙色）、语文（红色）、英语（蓝色）
- 封面展示科目名称 + 年份 + "龙老师官方"标识
- 卡片右下角显示"视频讲解 >"引导文字

### 数据说明

> **学习人数**：展示格式如"3.2万人学过""4648人学过"，超过 1 万时显示"X.X万"
> **推荐标签**：由后端标记，非所有试卷都有
> **试卷排序**：推荐试卷优先，同级别按学习人数降序

### 异常场景

| 场景 | 处理方式 |
|------|---------|
| 列表加载中 | 显示骨架屏占位 |
| 列表加载失败 | 显示加载失败提示 + 重试按钮 |
| 试卷封面图加载失败 | 使用科目对应的默认渐变色封面 |

### 关联模块

- → 视频讲解详情页（点击卡片跳转）
- ← 首页试卷讲解入口（进入来源）
