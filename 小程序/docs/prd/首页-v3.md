## 1. AI 龙老师对话入口卡片

### 功能说明

首屏核心模块，承载 AI 对话的三种入口：文字、语音、拍照。定位是"AI 即老师"——用户第一眼看到的是一个可以对话的 AI 老师，而非传统工具列表。

与 v1 的区别：v1 是工具导向（AI 入口和工具平级），v3 是对话导向（AI 入口在最顶部、最大面积）。

功能边界：本卡片只负责引导用户进入对话，不在首页内完成任何 AI 交互。

### 交互流程

1. **文字入口**：点击输入条任意区域 → 跳转对话页 → 自动弹出键盘，等待用户输入
2. **语音入口（微信风格 toggle）**：
   - 点击麦克风图标 → 输入栏切换为"按住说话"模式（麦克风图标变为键盘图标）
   - **长按"按住说话"**：激活语音识别 → 松开后自动识别 → 跳转对话页并发送识别文字
   - **短按"按住说话"**：直接跳转对话页 → 不弹出键盘，默认进入语音输入模式（显示按住说话按钮）
   - 点击键盘图标 → 切换回文字输入模式
3. **拍照入口**：点击相机图标 → 直接跳转拍照页（默认批改模式）

#### 为什么拍照不进对话页

拍照页底部已有模式切换（批改/搜题/录错题/错因诊断），用户无需先跟 AI 对话再选工具。如果用户想"拍照后跟 AI 聊"，可以在对话页内使用相机按钮。

### 业务规则

- 输入条 placeholder：`给 AI 龙老师发消息`，品牌导向而非功能描述
- 副标题轮播：每 3 秒切换一条，共 8 条（功能引导与百科问题交替），淡入淡出过渡
- 语音识别：demo 阶段模拟识别，从 5 条预设问题中随机返回

### 数据说明

> **副标题轮播内容**：8 条，交替展示功能引导（"拍一拍，作业秒批改"）和百科问题（"为什么天空是蓝色的？"）。后续可改为后端下发，支持个性化推荐。

### 异常场景

| 场景 | 处理方式 |
|------|---------|
| 长按录音中手指滑出按钮 | 取消录音，关闭浮层 |
| 语音识别中点击浮层外区域 | 取消识别，关闭浮层 |
| 长按不足 500ms 松开 | 视为短按，跳转对话页语音模式 |

### 关联模块

- → 对话页：文字和语音入口的目标
- → 拍照页：拍照入口的目标
- ← 对话页返回时回到本页

### 不做（本期）

- 真实语音识别 API 对接
- 个性化问候（如根据时段/姓名动态展示）
- 输入条支持直接打字（当前点击即跳转对话页）

## 2. 常用工具区

### 功能说明

首屏第二个模块，紧接 AI 卡片下方。采用"双大卡 + 8 宫格"的层级布局，按使用频率分两级展示。

选择这种布局而非 v1 的全平铺宫格，是为了制造视觉重心——作业批改和拍照搜题是最高频的工具，用大卡片突出；其余工具用小图标承载，避免首屏信息过载。

功能边界：工具区仅提供跳转入口，不承载任何工具的执行逻辑。

### 交互流程

1. **主功能大卡**：点击"作业批改"或"拍照搜题" → 跳转拍照页，自动切换到对应模式
2. **8 宫格工具**：点击任意工具 → 跳转对应功能页面

| 工具 | 跳转目标 |
|------|----------|
| 错题本 | 错题列表页 |
| 录错题 | 拍照页（录错题模式） |
| 错因诊断 | 拍照页（错因诊断模式） |
| 试卷擦除 | 试卷擦除页 |
| 试卷下载 | 试卷下载页 |
| 试卷讲解 | 试卷讲解页 |
| 小升初真题 | 试卷列表页（小升初类型） |
| 中考真题 | 试卷列表页（中考类型） |

### 业务规则

- 所有工具直接跳转功能页面，不经过 AI 对话页
- 用户通过 AI 对话页也可触达这些工具（AI 返回功能卡片），首页提供直达入口降低操作路径
- 工具入口无权限控制，所有用户可见

### 数据说明

> 工具列表固定 10 个，不依赖后端配置。

### 异常场景

| 场景 | 处理方式 |
|------|---------|
| 某个工具需下线 | 当前无动态控制机制，需前端发版 |

### 关联模块

- → 拍照页：作业批改、拍照搜题、录错题、错因诊断共用拍照页，通过模式参数区分
- → 各独立功能页：试卷擦除、试卷下载、试卷讲解、错题本等

### 不做（本期）

- 工具排序/显隐的后端配置化
- 各工具的使用次数角标
- 工具分组折叠

## 3. 视频课堂

### 功能说明

内容区域第一个模块，展示三种视频课程体系，支持按年级+学期筛选。一大两小卡片布局——"新课标重点讲"是重磅推荐内容，"校内同步课"和"拓展提升课"为常规入口。

### 交互流程

1. **年级+学期筛选**：点击标题右侧年级选择器 → 弹出筛选面板（上方"上/下学期"切换 + 下方 1-9 年级网格）→ 选择后关闭 → 按钮文案更新为"X年级上/下"
2. 点击任意课程卡片 → 跳转对应课程列表页，携带当前年级和学期参数

| 卡片 | 课程类型 | 角标 |
|------|----------|------|
| 大卡 | 新课标重点讲 | 重磅推荐 |
| 小卡左 | 校内同步课 | 无 |
| 小卡右 | 拓展提升课 | 无 |

### 业务规则

- 默认年级：取用户全局年级设置，无则默认"七年级"
- 默认学期：上学期
- 年级和学期切换后，课程内容按选中条件展示（后续对接后端数据）
- 大卡底部有学科标签（语文/数学/英语），仅展示不可筛选

### 数据说明

> 课程标题、描述、学科标签为固定内容。年级和学期筛选的 demo 阶段仅更新按钮文案，课程卡片内容不动态变化。

### 异常场景

| 场景 | 处理方式 |
|------|---------|
| 课程资源不存在 | 由课程详情页处理 |
| 点击年级弹窗外区域 | 弹窗关闭，年级不变 |

### 关联模块

- → 课程列表页
- ↔ 专项大招课、精品试卷：共享全局年级设置

### 不做（本期）

- 课程进度展示
- 根据年级+学期动态加载课程内容（当前仅筛选器 UI）
- 视频课程的收藏/点赞

## 4. 专项大招课

### 功能说明

按学科分类的专项课程卡片区域，支持按年级筛选。覆盖小学（1-6 年级）和初中（7-9 年级）两个学段。

### 交互流程

1. 点击右上角年级选择器 → 弹出 9 宫格年级面板（一～九年级），当前已选高亮
2. 选择年级 → 面板关闭 → 课程列表自动切换为对应学段内容
3. 点击课程卡片 → 跳转课程详情页

### 业务规则

- 默认年级：取用户全局年级设置，无则默认"七年级"
- 年级与学段的映射：1-6 年级 → 小学课程，7-9 年级 → 初中课程
- 小学 5 门课：语文古诗名篇、语文看图写话、语文阅读理解、英语语法提升、英语阅读专项
- 初中 5 门课：语文中文阅读、语文中文写作、语文名著专项、英语语法专项、英语题型技巧
- 2 列网格布局，卡片左侧彩色竖条区分学科（语文=绿色，英语=蓝色）

### 数据说明

> 课程数据固定，按学段分组。

### 异常场景

| 场景 | 处理方式 |
|------|---------|
| 点击年级弹窗外区域 | 弹窗关闭，年级不变 |

### 关联模块

- → 课程详情页
- ↔ 精品试卷：共享全局年级设置

### 不做（本期）

- 数学/科学学科的课程
- 课程节数、时长展示
- 课程完成进度

## 5. 精品试卷

### 功能说明

试卷推荐区域，提供三级联动筛选，覆盖一年级到九年级所有学段。筛选能力与 v1 完全对等。

### 交互流程

1. **一级（学科 Tab）**：点击"语文/数学/英语/科学"切换学科 → 三级筛选重置为"热门推荐" → 刷新列表
2. **二级（年级+学期）**：点击年级按钮 → 弹出面板（上方"上/下学期"切换 + 下方 1-9 年级网格）→ 选择后关闭 → 刷新列表
3. **三级（类型）**：点击"热门推荐/小升初真题/期末试题/期中试题/单元测试" → 刷新列表
4. **试卷点击**：跳转试卷详情页
5. **查看全部**：跳转试卷下载页

### 业务规则

- 学科切换时，年级和学期保持不变，类型重置为"热门推荐"
- 年级/学期切换时，类型保持不变
- 默认值：语文 + 七年级上 + 热门推荐
- 每个筛选组合最多展示 10 条
- 试卷角标：支持"推荐"（紫色）和"精选"（绿色）两种

### 数据说明

> 试卷数据按"学科 × 年级学期 × 类型"三维索引。每条试卷含标题、下载量、角标类型。全部为 mock 数据。

### 异常场景

| 场景 | 处理方式 |
|------|---------|
| 某筛选组合无数据 | 列表为空，无内容展示（当前无空状态提示） |
| 点击年级弹窗外区域 | 弹窗关闭 |

### 关联模块

- → 试卷详情页
- → 试卷下载页（"查看全部"入口）

### 不做（本期）

- 试卷收藏/下载计数实时更新
- 试卷搜索
- 按学校/地区筛选
- 空数据的引导提示

## 6. 导航与全局行为

### 功能说明

顶部导航栏、底部版本切换（仅 demo 用）、右上角演示/过稿切换、入场动画、跨页面返回逻辑。

### 交互流程

1. **顶栏"我的"Tab**：跳转个人中心
2. **顶栏历史记录图标**：跳转 AI 对话历史
3. **版本切换**（仅 demo）：在 v1/v2/v3 三个首页方案间切换
4. **演示/过稿切换**：点击"过稿"弹出产品稿浮窗；点击"演示"关闭浮窗

### 业务规则

- 所有二级页面返回时回到用户进入时的首页版本（v1 进入的回 v1，v3 进入的回 v3）
- 入场动画：模块按延迟分组从下方淡入
- 支持 URL 参数直接进入过稿模式

### 异常场景

| 场景 | 处理方式 |
|------|---------|
| 二级页面直接刷新（无浏览历史） | 兜底返回默认首页 |

### 关联模块

- → 个人中心
- → AI 对话历史
- ↔ 所有二级页面：维护返回路径

### 不做（本期）

- 版本切换上线时移除
- 底部 Tab 导航
- 页面切换过渡动画
