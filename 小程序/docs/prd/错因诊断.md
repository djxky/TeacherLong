# 错因诊断 产品稿

> 版本：v1.0 · 2026-04-15

---

## 1. 页面概述

### 功能说明

错因诊断页面展示拍照作业的原图，AI 自动识别错题并在原图上用红框标注，用户点击红框查看该题的全链路错因分析。支持两种入口：拍照诊断（整页作业）和错题本单题诊断。

与单纯的"对错判断"不同，错因诊断深入到解题思维链路，按"审题感知→知识唤醒→推演求解"三个维度逐一分析，帮助学生和家长理解**错在哪、为什么错、怎么改**。

功能边界：本页面只展示 AI 诊断结果，不支持在线修改答案或重新作答。

---

## 2. 拍照诊断结果页（整页模式）

### 功能说明

用户拍照后 AI 识别整页作业，将错题以红框标注在原图上，用户点击红框即可查看详细错因诊断。

### 交互流程

1. **进入方式**：用户在作业批改页面选择"错因诊断"模式拍照 → AI 识别完成后跳转本页面；或从历史记录页面点击错因诊断记录进入
2. **原图展示**：页面显示作业原图，AI 检测到的错题以红色边框标注在对应位置；仅标注错题，正确题目不标注
3. **点击热区**：点击红框 → 弹出底部诊断详情弹窗 → 展示该题的全链路分析
4. **再拍一张**：点击底部"再拍一张"按钮 → 返回拍照界面

### 业务规则

- 红框编号从 1 开始，按错题在页面上的位置顺序排列
- 仅错题有红框标注，正确题目不显示任何标记
- 如果整页无错题，不进入本页面（前置流程处理）

### 数据说明

> **错题热区坐标**：由 AI 识别返回，包含 left、top、width、height（百分比），对应原图中的位置
> **错题编号**：按顺序从 1 递增，仅统计错题

### 异常场景

| 场景 | 处理方式 |
|------|---------|
| AI 识别超时 | 前置流程处理，本页面不涉及 |
| 图片加载失败 | 显示加载失败占位图 + 重试按钮 |
| 无错题 | 前置流程拦截，显示"全部正确"结果页 |

### 关联模块

- ← 作业批改页（拍照入口）
- ← 历史记录页（查看历史诊断）
- → 错因诊断详情弹窗

## 3. 错因诊断详情弹窗

### 功能说明

底部弹出的详情面板，展示单道错题的完整错因分析，包括错因摘要、全链路诊断、下一步建议三个核心板块。

### 交互流程

1. **弹出**：点击原图上的错题红框 → 弹窗从底部滑出，覆盖页面 85% 高度
2. **题目切换**：弹窗内顶部显示圆形编号按钮 → 点击切换不同错题的诊断结果 → 弹窗内容刷新，滚动位置重置到顶部
3. **加入错题本**：点击"加入错题本"按钮 → Toast 提示"已加入错题本" → 按钮变为"已加入"不可再次点击
4. **再拍一页**：点击"再拍一页"按钮 → 返回拍照界面
5. **关闭弹窗**：点击顶部拖动条或右上角关闭按钮 → 弹窗收回，回到原图标注页面

### 业务规则

- 题目切换按钮仅在有多道错题时显示，单道错题时隐藏
- "加入错题本"按钮为一次性操作，点击后不可恢复
- 弹窗内支持上下滚动浏览完整诊断内容

### 数据说明

> **AI 错因摘要**：一段概括性文字，核心错误点以高亮标注（黄底+下划线）展示
> **全链路诊断维度**：按解题链路分为"审题感知""知识唤醒""推演求解"等维度，每个维度标记"掌握✓"或"待掌握✕"
> **待掌握维度**：额外展示错误详情（红色文字）和深度归因分析（绿色文字）
> **下一步建议**：包含总结文字 + 2 个 Tip 卡片

### 异常场景

| 场景 | 处理方式 |
|------|---------|
| 诊断内容为空 | Toast 提示"该题目正确，无需诊断" |
| 重复点击"加入错题本" | 按钮已变为"已加入"，不响应 |
| 弹窗打开时页面滚动 | 弹窗内部独立滚动，不影响页面滚动 |

### 关联模块

- ← 拍照诊断结果页（红框点击触发）
- → 错题本（加入错题本操作）
- → 拍照页（再拍一页）

## 4. 单题诊断页（来自错题本）

### 功能说明

独立全屏页面，从错题本进入，展示单道错题的完整错因诊断。与弹窗模式内容一致，但采用卡片式纵向布局，更适合沉浸式阅读。

### 交互流程

1. **进入方式**：从错题本页面点击"错因诊断"按钮进入 → 携带题目 ID 参数
2. **浏览**：依次展示错题原图、题目内容、AI 错因摘要、全链路诊断、举一反三建议
3. **返回**：点击底部"返回错题本"按钮或顶部返回箭头 → 返回错题本页面

### 业务规则

- 仅当错题有学生原始作答时，错题本中才显示"错因诊断"入口
- 页面独立于拍照诊断流程，不依赖整页作业的上下文

### 数据说明

> **题目 ID**：通过 URL 参数传入，用于查询对应的诊断数据
> **页面内容**：与弹窗模式的数据结构一致，仅展示形式不同

### 异常场景

| 场景 | 处理方式 |
|------|---------|
| 题目 ID 无效 | 自动返回错题本页面 |
| 无诊断数据 | 显示空状态提示 |

### 关联模块

- ← 错题本页面（入口）
- → 错题本页面（返回）
