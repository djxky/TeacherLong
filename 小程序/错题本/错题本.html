1. 页面概述

功能说明

错题本是学生在作业批改后，自动收录和手动上传错题的管理工具。支持按学科和题型筛选、查看题目详情（含拍照原图、题目文字、讲解视频、答案解析），以及批量管理（删除）和打印错题。

核心价值：帮助学生集中复习错题，避免重复犯错。

功能边界：错题本只做错题的"收集-浏览-管理-打印"，不做在线答题和重新作答。

入口

从首页点击"错题本"入口进入。

整体流程

进入错题本 → 按学科/题型筛选 → 点击卡片查看详情 → 查看 AI 推荐练习 / 查看错因诊断（仅数学） → 上传新错题 / 管理已有错题 / 打印错题

2. 错题列表页

功能说明

以时间倒序展示所有错题，每道错题以图片卡片形式呈现。支持按学科 Tab 和题型胶囊筛选。

页面结构

1. 顶部导航栏：返回箭头 + 标题"错题本"

2. 学科 Tab 栏：横向滑动，包含"数学、语文、英语、科学、物理、化学、历史、道法、地理、生物、其他"。默认选中"数学"。仅在有错题数据时展示。

3. 题型筛选行：毛玻璃胶囊按钮，"全部"固定展示，其余按当前学科动态展示（数学：单选题/填空题/解答题；英语：阅读理解/写作/语法 等）。仅在当前学科有错题时展示。

4. 错题卡片列表：纵向滚动，每张卡片包含题目拍照图片（120px 高）+ 底部信息行（日期时间）

5. 底部操作栏：吸底毛玻璃，包含"管理"按钮（左侧）+ "上传错题"按钮 + "打印错题"按钮

交互流程

1. 切换学科：点击学科 Tab → 切换到该学科 → 题型筛选重置为"全部" → 重新渲染列表

2. 筛选题型：点击题型胶囊 → 过滤当前学科下该题型的错题

3. 查看详情：点击卡片 → 从右侧滑入详情页

4. 上传错题：点击底部"上传错题" → 跳转到作业批改页面的"录错题"模式

5. 打印错题：点击底部"打印错题" → 跳转到打印错题页面

业务规则

  • 错题按时间倒序排列，最新添加的排最前面
  • 学科 Tab 和题型筛选仅在有错题数据时展示，无任何错题时整个筛选区域隐藏
  • "管理"按钮仅在有错题时展示
  • 卡片底部信息行的日期格式：当天显示"今天 HH:MM"，昨天显示"昨天 HH:MM"，更早显示"M月D日 HH:MM"

数据说明

  • 每道错题包含字段：题目 ID、学科、题型、拍照图片、收录日期、题目文字、讲解视频（可选）、答案解析、是否有学生原始作答、是否已生成错因诊断
  • 数据来源：作业批改后自动收录 + 用户手动拍照上传

异常场景

场景占位图主文案副文案操作
无任何错题（整体空状态）文件图标 · 48px 灰底圆还没有错题记录完成作业批改后，错题会自动收录按钮「去做作业」
当前学科无错题搜索图标 · 48px 灰底圆该科目暂无错题切换其他科目看看—
图片加载失败图片错误图标 · 灰色占位加载失败—点击重试

3. 管理模式

功能说明

进入管理模式后可批量选择错题并删除。不支持批量移动或归类。

交互流程

1. 进入：点击底部"管理"按钮 → 按钮文字变为"完成" → 每张卡片左上角出现圆形复选框 → 底部操作栏替换为"全选"+"删除"

2. 选择：点击卡片复选框 → toggle 选中态（蓝底白勾）；点击"全选" → 所有卡片复选框选中

3. 删除：点击"删除" → 选中的卡片缩小淡出消失（0.3s）→ Toast "删除成功" → 自动退出管理模式

4. 退出：点击"完成"按钮 → 清除所有选中态 → 恢复正常底部栏

业务规则

  • 管理模式下点击卡片不触发详情页跳转，只能点复选框
  • 未选中任何题目时点击"删除"，Toast 提示"请先选择要删除的题目"
  • 管理模式下底部Tab导航隐藏，操作栏上移
  • 切换学科时自动退出管理模式

异常场景

场景占位图主文案副文案操作
未选择题目点击删除—请先选择要删除的题目—Toast 提示，自动消失
删除后当前学科无题目搜索图标 · 48px 灰底圆该科目暂无错题切换其他科目看看—

4. 题目详情页

功能说明

全屏子页面，从右侧滑入。展示单道错题的完整信息：拍照原图、题目文字、AI 推荐练习（仅展示题目）、讲解视频、答案解析，以及错因诊断入口（仅限已开放学科）。

页面结构

1. 顶部导航栏：返回箭头 + 标题"题目详情" + 更多按钮

2. 拍照原图：圆角卡片展示题目照片

3. 题目内容：毛玻璃卡片，标题"题目内容"（紫色小标签）+ 题目文字（保留换行和排版）

4. AI 推荐练习：毛玻璃卡片，标题"✦ AI 推荐练习"。展示 AI 根据当前错题推荐的同类型练习题，仅展示题目不展示答案

5. 讲解视频：毛玻璃卡片，标题"讲解视频" + 视频播放器。无视频时展示占位图 +「暂无讲解视频」

6. 答案解析：毛玻璃卡片，标题"答案解析" + 解析文字

7. 错因诊断入口：吸底按钮，仅当题目所属学科属于已开放诊断学科时展示。按钮文字根据状态动态变化（见 4.2 节）

交互流程

1. 进入：从列表页点击卡片 → 详情页从右侧滑入

2. 浏览：纵向滚动查看各个内容板块

3. 播放视频：点击视频区域 → 原生播放器播放

4. 错因诊断：点击底部吸底诊断按钮 → 跳转到错因诊断页面

5. 返回：点击左上角返回箭头 → 详情页向右滑出

数据说明

  • 题目内容：原始题目文字，保留数学公式和选项格式
  • 讲解视频：可选字段，无视频时展示占位图 +「暂无讲解视频」
  • 答案解析：解题思路和答案说明

异常场景

场景占位图主文案副文案操作
无题目文字文字图标 · 灰色占位暂无题目文字——
无讲解视频视频图标 · 灰色占位暂无讲解视频——
无答案解析解析图标 · 灰色占位暂无答案解析——
视频加载失败视频错误图标 · 灰色占位视频加载失败—点击重试

4.1 AI 推荐练习

在题目详情页展示 AI 根据当前错题推荐的同类型练习题，帮助学生举一反三巩固知识点。仅展示题目，不展示答案，鼓励学生独立思考。

展示位置：位于"题目内容"卡片下方，以独立毛玻璃卡片呈现。

卡片结构：

  • 标题行："✦ AI 推荐练习"
  • 练习题列表：每道题包含题号（如"第1题"）+ 题目内容文字
  • 仅展示题面和选项，不展示答案

业务规则：

  • 根据当前错题的知识点和题型自动生成，数量 3-5 道
  • 与原错题难度相近
  • 推荐练习区域始终展示，无需用户手动触发

4.2 错因诊断入口

对于已开放诊断的学科（当前仅数学），在详情页底部以吸底按钮形式展示。根据是否已生成诊断结果，区分两种状态：

  • 已生成诊断：按钮文字"查看错因诊断"
  • 未生成诊断：按钮文字"生成错因诊断"

导航逻辑：

1. 点击诊断按钮 → 跳转到错因诊断页面，携带参数：来源=错题本、题目 ID、是否已分析

2. 已分析的题目 → 诊断页面直接渲染结果（无 loading）

3. 未分析的题目 → 诊断页面先展示约 20 秒 loading 动画，完成后展示结果

业务规则：

  • 非已开放学科的题目不展示任何诊断入口
  • 是否已生成诊断由后端返回的标记字段决定
  • 当前仅开放数学，后续视 AI 准确率逐步扩展

5. 上传错题

功能说明

用户手动拍照上传错题到错题本。点击"上传错题"后跳转到作业批改页面的"录错题"模式，复用其拍照 → 智能识别 → 手动框题 → 确认上传的完整流程。

交互流程

1. 入口：点击底部"上传错题"按钮

2. 跳转：携带录错题模式参数跳转到作业批改相机页面

3. 拍照/选择：在相机页面拍照或选择图片

4. 智能框题：系统自动识别题目区域，展示蓝色虚线框。支持多选（点击 toggle）

5. 手动框题（可选）：若自动识别不准确，可手动拖拽绿色框选定区域

6. 上传：点击上传按钮 → Loading 1.5s → Toast "成功添加 N 道错题" → 返回错题本并刷新列表

业务规则

  • 自动识别的题目框支持多选，手动框题的框也可叠加选择
  • 至少选中一个题目区域才能上传（否则上传按钮禁用）
  • 新上传的错题默认归属到当前选中的学科

6. 打印错题

功能说明

将错题整理为可打印的格式。统一入口为列表页底部的"打印错题"按钮。

交互流程

1. 点击底部"打印错题"按钮 → 弹出打印选项弹窗

2. 选择打印模式：

  • 原错题：仅打印选中的错题
  • 原错题 + 推荐练习：打印错题及其对应的 AI 推荐练习题

3. 确认后进入打印预览页面

业务规则

  • 打印内容为当前学科下的错题数据
  • 打印排版和预览在打印页面内完成

  • 错因诊断的结果会更新错题的"已诊断"状态
  • 打印功能读取错题本的当前数据及推荐练习

8. 不做（本期）

  • 除数学以外的学科错因诊断（后续视 AI 准确率逐步开放）
  • AI 推荐练习的在线作答和提交
  • 错题的手动编辑（修改题目文字、答案等）
  • 错题分组/标签/文件夹管理
  • 错题的导出（PDF / 图片）
  • 错题的分享功能
  • 错题的搜索功能
  • 错题的排序切换（当前固定时间倒序）
  • 错题的批量移动到其他学科
  • 基于错题数据的统计分析（薄弱知识点、错误趋势等）
